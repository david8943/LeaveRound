### 서비스 소개
사용자 계좌의 잔돈을 자동으로 기부하는 서비스
- 자동기부
사용자의 잔고를 설정된 단위로 나눈 후 남은 금액을 기부
- 이벤트(민들레)
사용자의 위치 기반으로 생성된 민들레 홀씨를 획득하여 기부할 수 있는 이벤트
